# REACT PORTFOLIO

![github](https://img.shields.io/badge/license-Apache-brightgreen.svg)

### Brief Overview
**REACT PORTFOLIO** is a portfolio on the react framework.
        

Features:
* component based architecture
* basic routing
* contact form with validation

### Setup

- `npm install`

### Running

- `npm start`

### Technologies

```    
    "react": "^17.0.2",
    "react-dom": "^17.0.2",
    "react-router-dom": "^5.2.0",
    "react-scripts": "4.0.3",
```
